SOURCE_NOTES for data files in this directory
==============================================

This directory contains self-hosted blocklist and whitelist data used by the
BlockNSFW extension as a remote sync source. The `codepurse/BlockNSFW`
GitHub repository is owned by the same maintainer who publishes this
extension, so the data and the data license are under the maintainer's
control.

Files
-----

- HOSTS.txt     Domain blocklist in standard hosts-file format (0.0.0.0 <domain>).
                Sourced from a one-time snapshot of the upstream
                codepurse/BlockNSFW HOSTS.txt (commit bee0db2, fetched 2026-06-03).
- WHITELIST.txt Global whitelist of domains that must never be blocked.
                Sourced from a one-time snapshot of the upstream
                codepurse/BlockNSFW WHITELIST.txt (commit bee0db2, fetched 2026-06-03).
