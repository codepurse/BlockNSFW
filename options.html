DNS Protection New

Block adult domains at the DNS level using Cloudflare for Families. This adds an extra layer of protection by checking domains against a constantly-updated global filter before the page even loads.

Enable DNS Filtering Query Cloudflare's family-safe DNS to detect and block adult domains automatically
🌐 DNS: Off
How it works

When enabled, each new domain you visit is checked against Cloudflare's family-safe DNS (1.1.1.3) via DNS-over-HTTPS. Results are cached locally so repeat visits have zero delay. This uses a free public service — no account or API key required.

Disable Instagram Reels Hide reel entry points and redirect direct reel pages back to the main Instagram feed
🔎 Safe Search: Off 📵 FB Reels: Off 📵 IG Reels: Off
How it works

Uses Chrome's declarativeNetRequest engine to rewrite search URLs (safe=active, adlt=strict, etc.) before the request leaves your browser. YouTube 18+ videos, Facebook Reels and Instagram Reels are blocked locally with an early page redirect plus DOM filtering. Nothing is sent to a third party.

For cookie-backed engines (Qwant, AOL Search, Presearch), BlockNSFW additionally pins the SafeSearch cookie to strict on every response and silently redirects the engine's own settings page back to its home, so the toggle cannot be flipped off from the UI.
